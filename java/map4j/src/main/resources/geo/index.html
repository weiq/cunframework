Map4J 是一个极简的JavaScript库(参考了Leaflet,及OL3等),并且集成了一个功能强大的地图查看工具(参考了GMap.Net),同时支持移动平台APP的地图包装. 它有你需要的每一个 功能.

Map4J 的设邓初衷是为了提供一个 simplicity, performance 和 usability 的前端API, 以及一个能 browser，及 cache任意地图瓦片的后端程序. 使用说明请见JavaScript API 若想获得源码请到 source code, 欢迎对它进行补充和完善.

我们创建了一个地图 ,绑定到id值为'map'的div上, 点击地图中的marker可以弹出popup窗口

	
    var map = T.map('map').setView([126.7023, 45.7441], 12);
    
    var map4J = new T.map4JTile(
    	'http://192.168.138.25:8080/geo/google/road/L{z}/R{y}/C{x}.png');
    
    map.addLayer(map4J);
    
    T.marker([51.5, -0.09]).addTo(map)
	    .bindPopup('A pretty CSS3 popup.<br> Easily customizable.')
	    .openPopup();

参考Google Map加载方式,在页面中只需引入一个地图核心文件<script src="http://ip/api?v=2.0"></script>, 无需手动引入其它脚本文件,可指定加载指定版本API或自动加载最新版本.避免了一个库引入多份程序
